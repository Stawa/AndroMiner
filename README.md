# AndroMiner Miner Binaries

This branch is generated by the Android Miner Binary workflow on `main`.

The app can download either binary:

```text
lib/arm64-v8a/tls/libxmrig.so
lib/arm64-v8a/notls/libxmrig.so
```

`lib/arm64-v8a/libxmrig.so` is kept as the TLS default for compatibility.

Source and build scripts live in the `miner-builder/` directory on the `main` branch.
